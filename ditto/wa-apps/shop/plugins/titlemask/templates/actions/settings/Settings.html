Title Mask

Правила, заданные в настройках плагина, работают только в том случае, если title и description не заданы для товара, категории, страницы.
Таким образом, к примеру, вы можете сделать, чтобы у всех товаров заголовок формировался по правилу, а у отдельных товаров был персональный заголовок.
Чтобы "выключить" правило, просто удалите его (оставьте пустую строку) и нажмите сохранить.

Название товара маленькими буквами

Включить возможность задавать отдельные правила для каждой категории






%product|lower% — название товара маленькими буквами
%category|lower% — название категории маленькими буквами


Если галочка включена, то в настройках категорий появится два поля, где вы сможете задать маску title и description для товаров в этой категории
Доступные переменные:
%product% — название товара
%price% — цена
%sku% — артикул товара
%summary% — краткое описание товара
%category% — название категории
%category_full% — полное название категории вместе с родительскими категориями
%parent_category% — название родительской категории
%root_category% — название главной родительской категории
%feature.CODE% — значение характеристики товара с кодом CODE, например %feature.brand%
%store_name% — название магазина (из настроек)
%store_phone% — телефон магазина (из настроек)

Название категории маленькими буквами

Заменять переменные в заданных явно мета-тегах






%category|lower% — название категории маленькими буквами


Если галочка включена, то заданные для категорий мета-теги воспринимаются как правила и там можно применять переменные плагина.
Доступные переменные:
%category% — название категории
%category_full% — полное название категории вместе с родительскими категориями
%parent_category% — название родительской категории
%count% — количество товаров в категории
%store_name% — название магазина (из настроек)
%store_phone% — телефон магазина (из настроек)
%page_num% — номер страницы
чтобы часть правила добавлялась только для страниц, начиная со второй, используйте квадратные скобки, например, так: [ - Страница %page_num%]
